Common questions

Do I need to install anything before opening the notebook?
No — the notebook's first cell installs the two packages it needs (pyodbc, pandas). Just open it and start running cells.
Do I need a password?
No. The notebook uses your Windows login automatically (Trusted_Connection=yes). If a cell ever asks you for a password, stop — that's a bug, screenshot and send to me.
Does this need VPN?
No — you're already on the VDI, so the VDI is inside the network.
How long should this take?
2–5 minutes including the kernel restart. If you've been at it for 15 minutes, stop and message me on Teams.
Can I edit the code if something looks off?
Please don't — the code is deliberately simple so we know exactly what's being tested. If anything's wrong, the notebook will say so itself.
The cell ran but nothing seems to have happened.
Look for In [*] on the left of the cell — that means it's still running. Wait. If it's been more than 30 seconds and nothing's printed, screenshot and send.
