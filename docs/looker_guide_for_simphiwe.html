Looker Studio — Click-by-Click Guide

For Simphiwe. Every single click. Nothing assumed. Follow top to bottom.

Time needed: About 3 hours. Take breaks.

IMPORTANT: When this guide says "select a data source", Looker Studio shows the FULL BigQuery path in the dropdown — something like fmn-sandbox.marts.v_dashboard_overview. Look for the last part of the name (after the last dot). That's how you find the right one. For example, if I say "pick v_dashboard_overview", look for the dropdown option that ENDS with v_dashboard_overview.

Step A: Create the report

1Open Looker Studio

  1. Open Chrome
  2. Type in the address bar: lookerstudio.google.com and press Enter
  3. Sign in with your FNB Google account (the one you use for BigQuery)
  4. You will see a home page. Look for a button that says + Blank Report — it might also be a big "+" icon or say "Create" → "Report"
  5. Click it
  6. A popup will appear asking you to add data. Leave it open — we'll use it in the next step

Step B: Add all data sources

You need to connect 18 BigQuery tables. You do this once. After that, every page can use them.

2Add the first data source

The "Add data to report" popup should be showing. If it closed, click Add data in the toolbar at the top of the screen.

  1. You will see icons for different connectors (Google Sheets, BigQuery, CSV, etc.)
  2. Click the one that says BigQuery (it has a hexagon icon)
  3. On the left side, you will see a list. Look for and click My Projects
  4. You will see project names. Click fmn-sandbox
  5. You will see dataset names. Click marts
  6. You will see a list of tables and views. Find v_dashboard_overview and click it
  7. A blue button ADD will appear at the bottom right. Click it
  8. A popup asks "You are about to add data to this report". Click ADD TO REPORT
  9. A table might appear on the canvas. Ignore it for now.

3Add the remaining 17 data sources

Repeat the exact same process for each table below. Each time:

  1. Click Add data in the toolbar at the top
  2. Click BigQuery
  3. Click fmn-sandbox → marts
  4. Find the table name in the list and click it
  5. Click ADD → ADD TO REPORT

Step C: Clean up

4Delete the auto-generated tables on the canvas

Every time you added a data source, Looker Studio may have dropped a random table on the canvas. Delete them all:

  1. Click on any chart or table sitting on the white canvas
  2. Press the Delete key on your keyboard
  3. Repeat until the canvas is completely blank (white with nothing on it)

5Rename the first page

  1. Look at the very bottom of the screen. You will see small tabs (like sheet tabs in Excel)
  2. You should see one tab — it says "Page 1" or "Untitled Page"
  3. Right-click on that tab (two-finger click on Mac)
  4. A menu appears. Click Rename
  5. Type: Overview
  6. Press Enter

Page 1: Overview

6Add a title text

  1. Look at the top menu bar. Click Insert
  2. A dropdown menu appears. Click Text
  3. Your cursor changes to a crosshair. Click and drag on the canvas to draw a rectangle at the top (about the width of the page, about 40px tall)
  4. A text box appears. Type: Executive Overview
  5. Select the text you just typed (Ctrl+A or Cmd+A)
  6. Look at the right side panel. You should see font options. Set size to 24 and click the B button for bold

7Add a scorecard (Transactions)

A scorecard shows a single big number — like "2.24B".

  1. Click Insert (top menu)
  2. Click Scorecard (it might be under "Add a chart" depending on your version — look for an icon that looks like a single number)
  3. Click and drag on the canvas to draw a small rectangle (about 200px wide, 80px tall) below the title
  4. A scorecard appears with some default number
  5. Look at the right side panel. You will see two tabs: Data and Style. Make sure you're on the Data tab
  6. Find "Data source". It shows a dropdown. Click it and look through the list for the one that ends with v_dashboard_overview. Click it.

How do I find the right data source in the dropdown?

The dropdown shows the full path like fmn-sandbox.marts.v_dashboard_overview. Just look for the name at the END. If you see ...v_dashboard_overview, that's the right one.

  1. Below the data source, find "Metric". Click the field name that's showing (or the little pencil icon next to it)
  2. A list of available fields appears. Find and click total_transactions
  3. Next to the field name, you should see a dropdown that says the aggregation (like SUM, AVG, COUNT). Make sure it says SUM. If not, click it and change it to SUM.
  4. Now switch to the Style tab (click "Style" at the top of the right panel)
  5. Find font size and set it to 28
  6. Find "Compact number" and check the box. This turns 2237540286 into "2.2B"

8Add 4 more scorecards (copy-paste method)

Instead of building each one from scratch:

  1. Right-click the scorecard you just made
  2. Click Copy (or press Ctrl+C / Cmd+C)
  3. Click somewhere empty on the canvas
  4. Right-click → Paste (or Ctrl+V / Cmd+V)
  5. Drag the pasted scorecard to the right of the first one
  6. Click the pasted scorecard. In the right panel → Data tab → click the metric field
  7. Change it to the next field from this list:
Scorecard labelChange metric to this field
Customerstotal_customers
Segmentedsegmented_customers
Churn scoredchurn_scored

Repeat: paste → change metric → drag into position. You should have 5 scorecards in a row.

9Add segment donut chart

  1. Insert → Pie chart
  2. Draw it on the canvas below the scorecards (about 300×250px)
  3. Right panel → Data tab
  4. Data source: click the dropdown and find the one ending with v_dashboard_segments
  5. Dimension: click the dimension field → find and select segment_name
  6. Metric: click the metric field → find and select customers. Make sure aggregation = SUM
  7. Switch to Style tab
  8. Look for "Donut" or "Inner radius" → set to 60% (this makes a hole in the middle)

Page 2: Client Pitch

This is the most complex page. Take your time.

10Create a new page

  1. Look at the bottom of the screen where the page tabs are
  2. Find the + icon (it's to the right of the existing tabs)
  3. Click it — a new blank page appears
  4. Right-click the new tab at the bottom → Rename → type Client Pitch → press Enter

11Add Category dropdown

  1. Click Insert (top menu)
  2. Look for Drop-down list (it might be under "Add a control" — look for a dropdown icon)
  3. Draw a rectangle at the top left of the page (about 200px wide, 36px tall)
  4. Right panel → Data tab
  5. Data source: find the one ending with v_dashboard_client_pitch
  6. Control field: click it → select CATEGORY_TWO

12Add Client dropdown

  1. Insert → Drop-down list
  2. Draw it next to the Category dropdown
  3. Right panel → Data source: find the one ending with v_dashboard_client_pitch (SAME one as the Category dropdown)
  4. Control field: select DESTINATION
CRITICAL: Both dropdowns MUST use the same data source — the one ending with v_dashboard_client_pitch. If they use different sources, selecting a category will NOT filter the client list.

14Add competitor benchmark bar chart

  1. Insert → Bar chart
  2. Draw it below the scorecards (about 400×250px)
  3. Right panel → Data source: find the one ending with v_dashboard_client_pitch
  4. Dimension: select DESTINATION
  5. Metric: select market_share_pct, aggregation = AVG
  6. Sort: in the right panel, scroll down and find "Sort". Click the field → select market_share_pct → set to Descending
  7. Switch to Style tab → look for a checkbox or toggle that says "Horizontal" → turn it ON (this makes bars go sideways)
  8. Set bar color to #0f172a

Duplicate this chart (right-click → Copy → Paste → drag next to it). Change the metric to spend_per_customer.

15Add store loyalty chart

  1. Insert → Bar chart (horizontal)
  2. Right panel → Data source: find the one ending with mart_store_loyalty
  3. Dimension: DESTINATION
  4. Metric: pct_loyal_50 (AVG)
  5. Sort: pct_loyal_50 descending

16Add time patterns table

  1. Insert → Table
  2. Right panel → Data source: find the one ending with mart_store_time_patterns
  3. Dimension: DESTINATION
  4. Metrics — you need to add MULTIPLE metrics. After adding the first one, look for a + Add metric button below it:
    • pct_early_morning (AVG)
    • pct_mid_morning (AVG)
    • pct_afternoon (AVG)
    • pct_evening (AVG)
    • pct_weekend (AVG)
  5. Style tab → look for "Heatmap" → turn it ON (colors cells based on values)

17Add trend line chart

  1. Insert → Time series chart (looks like a line chart with a clock)
  2. Right panel → Data source: find the one ending with mart_monthly_trends
  3. Date dimension: month
  4. Breakdown dimension: DESTINATION (this creates one line per store)
  5. Metric: total_spend (SUM)

18Add 3 demographics charts

All 3 use the data source ending with mart_demographic_summary:

  1. Age: Insert → Bar chart. Dimension: age_group. Metric: customers (SUM)
  2. Gender: Insert → Pie chart. Dimension: gender_label. Metric: customers (SUM). Style → Donut 60%
  3. Income: Insert → Bar chart. Dimension: income_group. Metric: total_spend (SUM)

19Add geography bar chart

  1. Insert → Bar chart (horizontal)
  2. Data source: find the one ending with mart_geo_summary
  3. Dimension: PROVINCE
  4. Metric: total_spend (SUM)
  5. Sort: total_spend descending
Add drill-down: In the Data tab, find "Drill down" toggle → turn it ON. You'll see PROVINCE listed. Click + Add dimension below it and select MUNICIPALITY. Now users can click a province bar to see its municipalities.

Page 3: Segments

20Create page + add content

  1. Click + at the bottom to add a new page → rename to Segments
  2. Dropdown filter: Insert → Drop-down list. Data source: v_dashboard_segments. Control field: segment_name
  3. Segment profiles table: Insert → Table. Data source: v_dashboard_segments. Dimension: segment_name. Metrics: customers (SUM), avg_spend (AVG), avg_txns (AVG), avg_recency_days (AVG), avg_merchants (AVG), pct_weekend (AVG)
  4. Retention curve: Insert → Line chart. Data source: mart_cohort_retention. Dimension: months_since_first. Metric: retention_pct (AVG). Style → line color #0f172a, show data points
  5. Momentum table: Insert → Table. Data source: mart_spend_momentum. Dimension: momentum_status. Metrics: Record Count, total_spend_12m (AVG), spend_change_pct (AVG), urgency_score (AVG)

Page 4: Churn & CLV

21Create page + add content

  1. New page → rename Churn & CLV
  2. 3 dropdown filters — all use data source: v_dashboard_churn
    • Control field: churn_risk_level
    • Control field: clv_tier
    • Control field: segment_name
  3. Churn drivers chart: Insert → Bar chart (horizontal). Data source: v_dashboard_churn. Dimension: reason_1. Metric: Record Count. Sort descending.
  4. CLV table: Insert → Table. Data source: v_dashboard_churn. Dimension: clv_tier. Metrics: Record Count, predicted_clv (AVG), historical_spend (AVG)
  5. Risk breakdown table: Insert → Table. Data source: v_dashboard_churn. Dimension: churn_risk_level. Metrics: Record Count, churn_probability (AVG), total_spend (SUM), days_since_last (AVG)

Page 5: Categories

22Create page + add content

  2. Dropdown: Data source: mart_category_scorecard. Control field: CATEGORY_TWO
  3. Health table: Data source: mart_category_scorecard. Dimension: CATEGORY_TWO. Metrics: total_customers, total_spend, growth_pct, avg_churn_pct, health_status

23Create page + add filters

  1. New page → rename Audiences
  2. Add a title text: Audience Marketplace (font size 20, bold)
  3. Add a subtitle text below: Pre-packaged audiences for activation via LiveRamp → Meta, Google, TikTok (font size 11, grey)
  4. Dropdown 1: Data source: mart_audience_catalog. Control field: audience_type. Label: "Type"
  5. Dropdown 2: Data source: mart_audience_catalog. Control field: top_province. Label: "Province"

24Add 3 scorecards

All use data source mart_audience_catalog:

  1. Total audiences: Metric = Record Count. Compact number OFF (it's a small number like 30)
  2. Total reachable: Metric = audience_size, aggregation = SUM. Compact number ON
  3. Avg audience size: Metric = audience_size, aggregation = AVG. Compact number ON

25Add audience catalog table

  1. Insert → Table (make it nice and wide, nearly full width of the page)
  2. Data source: mart_audience_catalog
  3. Dimensions: audience_name, audience_type
  4. Metrics (add ALL of these using the + button):
    • audience_size (SUM) — rename column header to "Reach"
    • avg_spend (AVG) — rename to "Avg Spend"
    • avg_age (AVG) — rename to "Avg Age"
    • pct_female (AVG) — rename to "% Female"
    • avg_income (AVG) — rename to "Avg Income"
    • top_province (leave as is)
    • top_segment (leave as is)
    • description (leave as is)
  5. Sort: audience_size descending (biggest audiences first)
  6. Style tab:
    • Turn on "Heatmap" on the audience_size column (darker = bigger audience)
    • Turn on "Show summary row" to see totals at the bottom
    • Set row height to "Comfortable" if available
    • Header font size: 10. Body font size: 11
To rename a column header: Click the table → Data tab → click the metric field → look for "Name" at the top → type the new name.

Anonymization page

27Duplicate Client Pitch for external use

  1. Right-click the Client Pitch tab at the bottom of the screen
  2. Click Duplicate page
  3. Right-click the new tab → Rename → Client Pitch (External)
  4. On this new page, click the competitor bar chart (the one showing market share)
  5. In the right panel → Data source dropdown → change from the one ending with v_dashboard_client_pitch to the one ending with v_pitch_external
  6. Then change the Dimension from DESTINATION to display_name
  7. Do the same for any other chart on this page that shows competitor names

Now: Internal page shows real names. External page shows "Competitor #1, #2, #3..."

Theme & finish

28Apply theme

  1. Click Theme and layout in the top menu bar
  2. Click Custom (or "Customize")
  3. Set:
    • Primary color: #1e3a5f
    • Secondary color: #2E75B6
    • Font: Roboto
    • Background: #fafbfc

29Test everything

  1. Click the eye icon at the top right of the screen — this switches to "View mode" (what other people will see)
  2. Go to Client Pitch page → change the Category dropdown → do the charts update?
  3. Change the Client dropdown → do the KPIs change?
  4. Go to each page → does every chart have data?
  5. If a chart is empty or shows an error, note which page and chart — tell Prosper
  6. To go back to editing, click the pencil icon at the top right

30Share the dashboard

  1. Click Share (blue button, top right)
  2. Add email addresses of the team
  3. Or set "Anyone in the organization with the link" → "Can view"
