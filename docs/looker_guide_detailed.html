Create the report

1Open Looker Studio

  1. Open Chrome. Go to lookerstudio.google.com
  2. Sign in with your FNB Google account (the one that has BigQuery access)
  3. You'll see a home page with "Recent reports" and a big + Blank Report button
  4. Click + Blank Report
  5. A popup says "Add data to report". We'll do this next.

Add data sources

You need to connect each BigQuery table/view as a "data source". You do this once. Every page shares them.

2Add your first data source

  1. The "Add data to report" popup should be showing. If not, look at the top menu bar and click Add data (it's a button with a + icon)
  2. You'll see a list of connectors (Google Sheets, BigQuery, CSV, etc.)
  3. Click BigQuery
  4. On the left side you'll see a list. Click My Projects (or "Shared Projects")
  5. Click fmn-sandbox (that's your project)
  6. Click marts (that's the dataset)
  7. You'll see a list of all tables and views. Click v_dashboard_overview
  8. Click ADD (blue button, bottom right)
  9. A popup says "You are about to add data to this report". Click ADD TO REPORT
  10. Done! You'll see a table appear on the canvas. Ignore it for now — we'll delete it later.

3Add the remaining data sources

Repeat the same process for each table below. Each time:

  1. Click Add data in the top menu bar (or Resource → Manage added data sources → Add a data source)
  2. Click BigQuery → fmn-sandbox → marts
  3. Select the table/view from the list
  4. Click ADD → ADD TO REPORT

Rename data sources

This makes it easier to find the right source later when building charts.

4How to rename

  1. In the top menu, click Resource
  2. In the dropdown, click Manage added data sources
  3. You'll see a list of all the sources you just added. They'll have names like fmn-sandbox.marts.v_dashboard_overview
  4. For each one, look on the LEFT side of the row — you'll see a pencil icon ✏️ (edit button). Click it.
  5. At the TOP of the screen that opens, you'll see the long name. Click on it — it becomes editable.
  6. Type the short name (from the table below)
  7. Click Done (top right)
  8. You're back at the list. Repeat for each source.

Clean up the canvas

5Delete the auto-generated tables

Every time you added a data source, Looker Studio may have dropped a table on the canvas. Delete them all:

  1. Click on any table/chart on the canvas
  2. Press Delete on your keyboard (or right-click → Delete)
  3. Repeat until the canvas is blank

6Rename the first page

  1. Look at the bottom of the screen — you'll see page tabs (like Excel sheets)
  2. Right-click the tab that says "Page 1" (or "Untitled Page")
  3. Click Rename
  4. Type Overview
  5. Press Enter

Page 1: Overview

7Add a title

  1. In the top toolbar, click Insert
  2. Click Text
  3. Draw a text box at the top of the canvas by clicking and dragging
  4. Type: Executive Overview
  5. Select the text. In the right panel under "Text Properties", set font size to 24 and weight to Bold

8Add the first scorecard (Transactions)

  1. Click Insert in the top toolbar
  2. Click Scorecard (it looks like a single number)
  3. Draw a small rectangle on the canvas (about 200px wide, 80px tall)
  4. A scorecard appears. Look at the right panel — this is where you configure it
  5. At the top of the right panel, you'll see Data source. Click the dropdown and select DS_Overview
  6. Below that, you'll see Metric. Click the dropdown/pencil next to it
  7. Select total_transactions
  8. Make sure the aggregation says SUM (click the dropdown next to the field name if you need to change it)
  9. In the right panel, scroll down to Style tab. Set:
    • Font size: 28
    • Check "Compact number" (this turns 2237540286 into 2.2B)

9Add the remaining 4 scorecards

The fastest way: right-click the scorecard you just made → Copy → Paste. Then change the metric:

ScorecardMetric field

To change the metric on a pasted scorecard:

  1. Click the pasted scorecard
  2. In the right panel → Data tab → click the metric field
  3. Select the new field from the list

Arrange all 5 in a row across the top of the page.

10Add segment donut chart

  1. Insert → Pie chart
  2. Draw it on the canvas (about 300x250px)
  3. Right panel → Data source: select DS_Segments
  4. Dimension: click the dimension field → select segment_name
  5. Metric: click the metric field → select customers. Set aggregation to SUM
  6. Switch to Style tab (right panel, second tab at top)
  7. Find "Donut" or "Inner radius" → set to 60% (this makes it a donut instead of a full pie)
  8. Under "Color" → click each segment and set:
    • Champions: #0f172a
    • Loyal High Value: #1e3a5f
    • Steady Mid-Tier: #4CAF50
    • At Risk: #FF9800
    • Dormant: #f44336

11Add churn headline scorecards

These show the "666K at risk / R31.6B spend" numbers.

  1. Insert → Scorecard
  2. Data source: DS_Churn
  3. Metric: total_spend (SUM)
  4. Now you need a filter to show only Critical + High risk:
    1. In the right panel, scroll down to Filter
    2. Click Add a filter
    3. Click Create a filter
    4. Name it: Critical + High only
    5. Set: churn_risk_level → Include → In → type Critical, press Enter, type High, press Enter
    6. Click Save
  5. The scorecard now shows spend at risk for Critical + High only

Copy this scorecard twice. Change one to metric = Record Count (this counts customers). Change the other to a calculated field for 10% recovery (or just manually label it).

Page 2: Client Pitch

12Create a new page

  1. Look at the bottom of the screen where the page tabs are
  2. Click the + icon (right side of the tabs) to add a new page
  3. Right-click the new tab → Rename → type Client Pitch

13Add Category dropdown filter

  1. Insert → Drop-down list (under "Add a control")
  2. Draw it at the top left of the page
  3. Right panel → Data source: DS_ClientPitch
  4. Control field: click it → select CATEGORY_TWO
  5. In the Style tab, you can set a label: type Category

14Add Client dropdown filter

  1. Insert → Drop-down list
  2. Draw it next to the Category dropdown
  3. Data source: DS_ClientPitch (SAME source — this is important so they cross-filter)
  4. Control field: DESTINATION
  5. Label: Client
Both dropdowns MUST use the same data source (DS_ClientPitch). If they use different sources, selecting a category won't filter the client list.

Anonymization (Internal vs External)

30Duplicate Client Pitch page

  1. Right-click the Client Pitch tab at the bottom
  2. Click Duplicate page
  3. Right-click the new tab → Rename → Client Pitch (External)
  4. On this new page, click each chart that shows competitor names (the benchmark bars, time patterns table, loyalty chart)
  5. For each chart: in the right panel, click the data source dropdown and change from DS_PitchInternal (or DS_ClientPitch) to DS_PitchExternal
  6. Then change the dimension from DESTINATION to display_name

Now: Internal page = real names. External page = "Competitor #1, #2, #3..."

To hide the external page during internal use: right-click tab → Hide in view mode. Unhide it when you need to present to a client.

Drill-downs

31Add drill-down to geography chart

  1. Click the geography bar chart on the Client Pitch page
  2. In the right panel → Data tab
  3. Find the Drill down toggle → turn it ON
  4. You'll see the dimension PROVINCE already there
  5. Click the + below it to add another dimension
  6. Select MUNICIPALITY
  7. Now when a user clicks a province bar, it drills into municipalities

Repeat this pattern for any chart where you want drill-down.

Theme & branding

32Set theme

  1. Click Theme and layout in the top menu
  2. Click Custom
  3. Set:
    • Primary color: #1e3a5f
    • Secondary color: #2E75B6
    • Font: Roboto (or Google Sans)
    • Background: #fafbfc

33Share the dashboard

  1. Click Share button (top right, blue)
  2. Add email addresses, or set to "Anyone in the organization with the link can view"
  3. For external clients: never share the link. Instead: File → Download as PDF.

You're done!
